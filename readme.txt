## Installation

To run this project, you need to have Python 3 and pip installed on your system. You can install Python from the official website: https://www.python.org/downloads/

After installing Python, you can set up the project environment by following these steps:

1. Clone the repository:
   ```bash
    git clone git@github.com:Muhammedsinanck/CloudQA_assignment.git

2. Navigate to the directory 
    ```bash 
    cd your-cloned-path

Note: If you want to run in seperate environment you can create your own enviornment to install requirements

3. Run the command 
    ```bash 
    pip install -r requirements.txt

3. Run the code 
    ```bash 
    python3 test.py

Hurray!!! You are done
